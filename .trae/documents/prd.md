## 1. 产品概述

三维医学影像 MPR 查看器是一款纯 Web 端的医学影像可视化工具，专为放射科医生设计，支持 DICOM 格式的三维体积数据浏览与多平面重建。

- 核心价值：无需安装客户端，通过浏览器即可实现专业级医学影像三维浏览
- 目标用户：放射科医生、医学影像技师、临床医生
- 解决问题：传统医学影像查看依赖厚重的桌面客户端，跨平台、远程会诊不便

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|----------|
| 放射科医生 | 无需注册，本地工具 | 导入 DICOM 数据、MPR 浏览、窗宽窗位调节、切层滑动 |

### 2.2 功能模块
1. **数据导入模块**：支持批量导入 DICOM 切片文件，自动识别序列
2. **三维重建模块**：WASM 加速构建 3D 纹理体积块
3. **MPR 查看模块**：轴状面/矢状面/冠状面三视图联动浏览
4. **交互控制模块**：切层滑动、窗宽窗位调节、缩放平移
5. **信息展示模块**：显示 DICOM 元数据、当前切层位置信息

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 主查看页面 | 顶部工具栏 | 导入 DICOM、视图切换、重置视图 |
| 主查看页面 | 三视图容器 | 轴状面、矢状面、冠状面三个视图同步显示 |
| 主查看页面 | 右侧控制面板 | 切层滑块、窗宽窗位调节、像素信息显示 |
| 主查看页面 | 底部状态栏 | 当前序列信息、切层索引、空间坐标显示 |

## 3. 核心流程

用户导入 DICOM 序列 → 系统解析 DICOM 文件 → WASM 构建三维体积纹理 → 初始化三个正交视图 → 用户滑动切层/调节窗宽 → 实时渲染对应切面

```mermaid

    A["导入 DICOM 文件"] --> B["解析 DICOM 元数据与像素"]
    B --> C["WASM 构建 3D 体积纹理"]

    D --> E["轴状面视图"]
    D --> F["矢状面视图"]
    D --> G["冠状面视图"]
    H["切层滑动交互"] --> D
    I["窗宽窗位调节"] --> D
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深灰蓝 (#1a1f2e) 背景，医疗专业感
- 强调色：青色 (#00d4ff) 高亮与交互元素
- 文字色：浅灰 (#e0e6ed) 主文字，中灰 (#8892a6) 次要文字
- 按钮风格：扁平化设计，细边框，悬停微动效
- 字体：使用 JetBrains Mono 等宽字体显示数值，Inter 作为界面字体
- 布局风格：三栏式布局，左侧/右侧控制面板，中央三视图网格
- 整体风格：医疗专业级深色主题，高对比度，减少视觉疲劳

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|--------|
| 主查看页面 | 顶部工具栏 | 导入按钮、视图布局切换、重置按钮、分隔线、应用标题 |
| 主查看页面 | 三视图容器 | 2x2 网格布局，三个视图 + 一个 3D 定位图 |
| 主查看页面 | 视图卡片 | 每个视图带标题标签、当前切层数显示、悬停高亮边框 |
| 主查看页面 | 右侧控制面板 | 分组折叠面板、滑块控件、数值输入框、信息表格 |
| 主查看页面 | 底部状态栏 | 左侧序列信息、右侧坐标信息、分隔符 |

### 4.3 响应式
- 以桌面端为主要设计目标（放射科医生使用工作站）
- 支持最小 1280px 宽度显示
- 三视图布局在小屏幕下自动切换为垂直堆叠
- 触摸设备支持双指缩放与滑动切层

### 4.4 3D 场景指引
- 环境：深色背景，无多余光源，专注医学影像本身
- 光照：采用影像自身灰度值作为亮度，不添加额外光照
- 相机：正射投影，每个视图对应一个正交观察方向
- 交互：鼠标滚轮缩放，左键拖动平移，滑块精确控制切层
- 后处理：支持窗宽窗位调节的实时对比度变换
- 性能目标：60fps 实时渲染，切层滑动无卡顿
